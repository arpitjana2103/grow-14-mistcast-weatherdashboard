It provides real-time weather conditions, a 48-hour hourly temperature chart, an 8-day daily forecast, and detailed atmospheric metrics - including UV index, wind compass, humidity, visibility, precipitation, sunrise/sunset arcs, and air quality (AQI) - all in one clean interface. ## Application Features

MistCast is a client-side weather dashboard built with React and the OpenWeatherMap API,
delivering real-time weather intelligence through a clean, visually rich interface.

**1. Complete Weather Intelligence**: Current conditions, a 48-hour interactive temperature chart, and an 8-day daily forecast — all in one seamless view.\
**2. Rich Detail Cards**: Eight dedicated mini-cards covering Sunrise/Sunset arc, Moonrise arc, Wind compass, UV Index, Humidity, Precipitation, Visibility, and Cloud cover.\
**3. Air Quality Index**: Full AQI breakdown with per-pollutant gradient scale indicators (SO₂, NO₂, PM2.5, PM10, O₃ and more).\
**4. Live Weather Map**: Leaflet-powered interactive map with switchable overlay layers — Temperature, Precipitation, Clouds, Pressure, and Wind.\
**5. Fully Personalized**: Search any city worldwide or detect your location, switch between Metric / Imperial units, and toggle Light / Dark theme — all persisted across sessions.\
**6. Responsive**: Optimized across all screen sizes — desktop, tablet, and mobile.

## Run Locally

**Step 1 :** Clone the repository

```bash
git clone https://github.com/arpitjana2103/grow-14-mistcast-weatherdashboard.git
cd grow-14-open-weather-cast
```

**Step 2 :** Install `npm packages`

```bash
npm install
```

**Step 3 :** Create a `.env.local` file in the project root with the following variable

```
VITE_OPENWEATHERMAP_API_KEY={{YourOpenWeatherMapAPIKey}}
VITE_LOCATIONIQ_API_KEY={{YourLocationIQAPIKey}}
```

> Get OpenWeather API key at [openweathermap.org](https://openweathermap.org/api). The app uses the **One Call API 3.0** and **Air Pollution API**.

> Get LocationIQ API key at [locationiq.com](https://locationiq.com/). The app uses **Autocomplete** & **Reverse Geocoding** API.

**Step 4 :** Start the development server

```bash
npm run dev
```
